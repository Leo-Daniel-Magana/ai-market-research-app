# AI Market Research App

An AI-powered market intelligence tool that generates structured competitive analyses for any industry or company.

## Live Demo
🔗 [leo-daniel-magana.github.io/ai-market-research-app](https://leo-daniel-magana.github.io/ai-market-research-app)

> **Note:** Live API integration currently in progress. Screenshots below show full functionality.

## Features
- Instant market analysis for any industry or company
- Structured output: Market Overview, Key Players, Trends, Opportunities, Risks, and Strategic Recommendation
- Clean, responsive dark-mode UI
- Built with vanilla HTML, CSS, and JavaScript

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **AI:** Google Gemini API
- **Hosting:** GitHub Pages
- **API Proxy:** Cloudflare Workers

## How It Works
1. User enters a market or industry query
2. The app sends the query to a Cloudflare Worker proxy
3. The Worker calls the Gemini API securely
4. Results are parsed and displayed in structured cards
